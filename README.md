**Overview**:
This project is a Power BI dashboard created using Amazon sales data to analyze sales performance, profit, quantity sold, and shipping costs across different markets, countries, and product categories.

**Tools Used**
- Power BI
- Power Query
- Excel Dataset

**Dashboard Features**
- KPI cards showing Total Sales, Profit, Quantity and Shipping Cost
- Sales analysis by Category
- Sales distribution by Market
- Sales comparison by Country and City
- Sales by Ship Mode
- Interactive filters for Category and Market

**Key Insight**
The dashboard helps identify top-performing markets, categories, and cities, enabling better understanding of sales trends.
